# Code of Conduct

## Our commitment

GuardianCI is an open, welcoming project. Everyone who participates — through issues, pull requests, discussions, or any other channel — is expected to treat others with respect and professionalism.

## Expected behaviour

- Be constructive and specific in feedback
- Assume good intent; ask for clarification before drawing conclusions
- Focus criticism on ideas and code, not on people
- Welcome contributors of all experience levels

## Unacceptable behaviour

- Personal attacks, insults, or dismissive language
- Public or private intimidation
- Sharing others' private information without permission
- Any conduct that would be considered unprofessional in a work setting

## Scope

This Code of Conduct applies to all project spaces: GitHub issues, pull requests, discussions, and any community channels associated with GuardianCI.

## Enforcement

Violations can be reported by opening a private [GitHub security advisory](https://github.com/noobbatman/-GuardianCI/security/advisories/new) or emailing the maintainer directly. Reports are reviewed promptly and kept confidential.

Maintainers may remove, edit, or reject comments, commits, and other contributions that do not align with this Code of Conduct, and may temporarily or permanently ban contributors for repeated or severe violations.
